JavaScript

синтаксис однострочного комментария
синтаксис многострочного комментария
Переменная состоит: из имени и выделенной области памяти, которая ему соответствует
Объявление переменной(создание в памяти компьютера) с помощью ключевого слова var
Инициализация переменной(присвоение значения переменной) с помощью оператора «присвоить» =
Название переменной первый символ (буква,_, $, но не число)
Зарезервированные имена (var, class, return, export и т.д) нельзя использовать для названия переменных
Константа
  • это переменная, которая никогда не меняется.
  • Как правило, их называют большими буквами, через подчёркивание. var COLOR_RED = «#F00";
  • Технически, константа является обычной переменной, то есть её можно изменить. Но мы договариваемся этого не делать.
Правила именования переменных:
  • английские буквы (стандарт);
  • верблюжья нотация (несколько слов пишутся слитно, каждое слово начинается с большой буквы) var camelCase
  • именование через подчеркивание var camel_Case
  • Имя переменной должно соответствовать хранимым в ней данным;
типы данных в js (первые пять типов - примитивные):
  1. Число «number»
  2. Строка «string»
  3. Булевый (логический) тип «boolean»
  4. Специальное значение «null»
  5. Специальное значение «undefined»
  6. Объекты «object»
Оператор typeof
  • синтаксис typeof(x) или typeof x
  • возвращает имя типа данных
Операторы:
  1. Операнд – то, к чему применяется оператор 5 * 2
  2. Унарным называется оператор, который применяется к одному выражению x+
  3. Бинарным называется оператор, который применяется к двум операндам y - x// унарный минус
  4. Особенность исключительно бинарного оператора «+». Если хотя бы один аргумент является строкой, то второй будет также преобразован к строке!
  5. Если в выражении есть несколько операторов – порядок их выполнения определяется приоритетом
  6. Оператор "=" возвращает(вычисляет) значение
  7. Оператор Взятие остатка %. a % b – это остаток от деления a на b.
  8. Оператор Инкремент ++ . увеличивает переменную на 1
  9. Оператор Декремент - - . уменьшает переменную на 1
  10. Инкремент/декремент можно применить только к переменной. Код 5++ даст ошибку.
  11. Постфиксная форма i++ отличается от префиксной ++i тем, что возвращает старое значение, бывшее до увеличения.
методы и свойства Все значения в JavaScript, за исключением null и undefined, содержат набор вспомогательных функций и значений, доступных «через точку». Такие функции называют «методами», а значения – «свойствами».

JavaScript (числа)

Infinity
  • это математическая бесконечность ∞
  • бывает +Infinity и -Infinity
  • Infinity больше любого числа
  • Деление на ноль не считается ошибкой в js. оператор возвращает Infinity
NaN (Not-A-Number)
  • NAN используется для обозначения математической ошибки
  • Если математическая операция не может быть совершена, то возвращается специальное значение NaN (Not-A-Number)
  • Значение NaN – единственное, в своем роде, которое не равно ничему, включая себя
  • Деление на ноль не считается ошибкой в js. оператор возвращает Infinity
проверка на NaN:
  • с помощью метода isNaN()
  • возвращает true, если аргумент равен NaN или является нечисловым значением, которое невозможно преобразовать в число(например, строка или объект)
NaN возвращается при:
  • при попытке деления бесконечность на бесконечность
  • при вычислении квадратного корня отрицательного числа
  • при невозможности преобразовать тип данных в число(например, строка)
